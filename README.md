# I<sup>2</sup>C - Inter Integrated Circuit

</center>

I<sup>2</sup>C - Inter Integrated Circuit is is a synchronous, multi-master/multi-slave, single-ended, serial communication bus, widely used for attaching lower-speed peripheral integrated circuits (ICs) to processors and microcontrollers in short-distance, intra-board communication. 

*Source : [Wikipedia](https://en.wikipedia.org/wiki/I2C)*
